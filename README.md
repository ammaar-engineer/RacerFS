# Introduction
RacerFS adalah adaptasi dari projek lama ku yang bernama RacerCLI
- https://github.com/ammaar-engineer/racer-cli-system
- https://github.com/ammaar-engineer/racer-backend-system
Kedua nya adalah projek RacerCLI. Saya membuat projek RacerCLI karena dulu saya mengalami kesulitan ketika ingin setup linux yang baru saya install. Akhir nya saya membuat RacerCLI dan projek nya pun selesai. Namun saya melihat projek RacerCLI saya kurang sempurna dan memiliki potensi untuk dikembangkan menjadi sesuatu yang lebih berguna. Akhir nya saya merubah design projek nya yang awal nya hanya untuk saya. Menjadi bisa digunakan oleh banyak orang.
****
# Fitur utama RacerFS
RacerFS memiliki fitur-fitur yang mirip dengan RacerCLI. Fitur-fitur nya yaitu
- Storage: untuk menyimpan file
- Snippet: sistem yang mirip alias pada bashrc
- AI: teman kecil untuk bantu setup. (Penyedia wadah untuk AI kerja di terminal)
****
# Keunikan RacerFS
Keunikan RacerFS ada dimana dia tinggal. RacerFS akan menjadi SaaS full berbasis terminal dengan TUI. Mengapa dibuat demikian? 
1. Saya gamau ribet
2. Tidak mungkin menggunakan web browser ketika linux belum memiliki DE dan masih dalam terminal kosong
Maka dari itu saya ingin menjadikan nya SaaS full berbasis terminal dengan TUI
